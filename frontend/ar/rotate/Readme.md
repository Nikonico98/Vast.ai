# rotate 目录说明 / rotate Folder Guide

本目录是旋转互动 AR 页面发布产物。
This folder contains the deployed build for rotate interaction AR.

## 用途 / Purpose

用户通过旋转 3D 物体触发剧情互动。
Users trigger story interaction by rotating a 3D object.
