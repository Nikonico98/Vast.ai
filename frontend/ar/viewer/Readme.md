# viewer 目录说明 / viewer Folder Guide

本目录是单模型 AR 预览页面发布产物。
This folder contains the deployed build for single-model AR preview.

## 用途 / Purpose

用于查看单个 3D 模型在现实空间中的叠加效果。
Used to preview one 3D model overlaid in real-world space.
